Neon Nights

Group Project

Description
An action role playing game similar to the diablo series of games. The player moves through a set of levels and gradually gets stronger as they level up. We worked hard trying to meet some deadlines but we managed to get it all out on time. I spent most of this project trying to keep everyone working, and it worked almost as i intended it to do. Though i had to spend more time than i expected on trying to keep the less experienced from falling behind. The AI was based on a system i developed myself, it made all AI behaviours really easy to add on and refine.

C#

Packet Loss

Group Project

Description
A game me and a couple of awesome devs made during a game jam. Spent most of the time reworking and adding details to the map. The game is all about putting the coloured boxes into the correct bins in order to earn as high of a score as possible. Each bin in the game is situated in a way to make the player see it in a different way. Another dev made some nice promotional paintings featuring the devs which we put into the level, i'm the third one from the left. Making this game was mighty fun and i'm even considering making an extended version of it with all the ideas we discarded.
